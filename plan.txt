Endpoints

1-Técnicos

POST /technicians → cadastrar técnico
GET /technicians → listar técnicos
GET /technicians/:id → detalhes de um técnico
PUT /technicians/:id → atualizar técnico
DELETE /technicians/:id → remover técnico

2-Equipamentos

POST /equipments → registrar entrada de equipamento
GET /equipments → listar equipamentos
GET /equipments/:id → detalhes de um equipamento
PUT /equipments/:id → atualizar status ou dados
DELETE /equipments/:id → remover equipamento

3-Ordens de Serviço

POST /service-orders → abrir ordem de serviço
GET /service-orders → listar ordens
GET /service-orders/:id → detalhes da ordem
PUT /service-orders/:id → atualizar ordem (ex: fechar)
DELETE /service-orders/:id → remover ordem

4-Peças Substituídas

POST /parts-replaced → registrar peça substituída
GET /parts-replaced/:service_order_id → listar peças de uma ordem
